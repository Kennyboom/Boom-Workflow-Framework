# Reference: Brand Voice + Copy Techniques

## Brand Voice Architecture

```
BRAND VOICE SPECTRUM:
│ Dimension          │ Position                              │
│ Formal ←→ Casual   │ [        ●              ]            │
│ Serious ←→ Fun     │ [           ●           ]            │
│ Technical ←→ Simple│ [      ●               ]             │
│ Reserved ←→ Bold   │ [               ●       ]            │
│ Classic ←→ Modern  │ [                  ●    ]            │

VOICE ATTRIBUTES (3 từ): [Word 1] [Word 2] [Word 3]

DO vs DON'T:
✅ "Tạo video chất lượng studio — trong 5 phút"
❌ "Phần mềm tạo video chuyên nghiệp"

TONE VARIATIONS:
│ Context   │ Tone          │ Example                    │
│ Homepage  │ Confident     │ "Tương lai bắt đầu ở đây" │
│ Error     │ Empathetic    │ "Oops! Để em sửa ngay"     │
│ Onboarding│ Encouraging   │ "Chỉ 3 bước, thế là xong!" │
│ Pricing   │ Transparent   │ "Rõ ràng. Không phí ẩn."   │
```

## Headline Formulas

```
1️⃣ BENEFIT-FIRST: "[Kết quả] — [Không cần X]"
2️⃣ QUESTION HOOK: "[Câu hỏi chạm pain point]?"
3️⃣ HOW-TO PROMISE: "Cách [kết quả] mà [bất ngờ]"
4️⃣ SOCIAL PROOF: "[Con số] [đã/đang] [hành động]"
5️⃣ CONTRAST: "Từ [trạng thái cũ] đến [trạng thái mới]"
6️⃣ CURIOSITY GAP: "[Statement bất ngờ]"

Rules: ≤12 từ | Có số | Cảm xúc > logic | Đọc to kiểm tra
```

## Premium Copy Techniques

```
1️⃣ POWER WORDS: Instantly, Effortless, Proven, Free, Unlock
   ❌ Tránh: Very, Really, Basically, Actually, Just

2️⃣ RHYTHM: Short. Punchy. Then a longer flowing sentence.

3️⃣ SPECIFICITY: ❌ "Tiết kiệm thời gian" → ✅ "Tiết kiệm 15 giờ/tuần"

4️⃣ SENSORY: ❌ "Giao diện đẹp" → ✅ "Mượt như lụa, sắc nét từng pixel"

5️⃣ CONTRAST PAIRS: "Ít hơn effort. Nhiều hơn impact."

6️⃣ RULE OF 3: "Nhanh hơn. Thông minh hơn. Đẹp hơn."

7️⃣ ANALOGY: "Blockchain như sổ cái chung — ai cũng đọc, không ai sửa."
```
