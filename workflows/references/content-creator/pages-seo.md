# Reference: Page Content + Microcopy + SEO

## Page-by-Page Content Map

```
🏠 HOMEPAGE:
□ Hero headline + subheadline
□ 3 key benefits (icons + text)
□ Social proof bar (logos / numbers)
□ How it works (3 steps)
□ Testimonial section
□ Final CTA section

📋 FEATURES PAGE:
□ Feature headline per section
□ Benefit-first description
□ Use case mini-stories
□ Comparison (before/after)

💰 PRICING PAGE:
□ Plan names (sáng tạo, không 'Basic/Pro/Enterprise')
□ Feature comparison clear
□ 'Most Popular' highlight
□ FAQ (objection handling)
□ Money-back guarantee

📖 ABOUT PAGE:
□ Origin story (StoryBrand)
□ Mission (1 câu)
□ Team (human, not corporate)
□ Values (3-5, có ý nghĩa)

📧 EMAIL SEQUENCES:
□ Welcome (warm, helpful)
□ Onboarding (3-5 emails)
□ Re-engagement
□ Newsletter template
```

## Microcopy Playbook

```
🔘 BUTTONS:
❌ 'Submit' → ✅ 'Tạo video của tôi'
❌ 'Sign Up' → ✅ 'Bắt đầu miễn phí'
❌ 'Download' → ✅ 'Tải về — chỉ 2MB'
❌ 'Learn More' → ✅ 'Xem cách hoạt động'

📝 FORM LABELS:
❌ 'Email Address' → ✅ 'Email (để gửi video cho bạn)'

⚠️ ERROR MESSAGES:
❌ 'Invalid input' → ✅ 'Hmm, email chưa đúng. Thử lại nhé!'
❌ 'Error 500' → ✅ 'Server đang bận. Thử lại sau 30 giây.'

✅ SUCCESS:
❌ 'Success' → ✅ 'Tuyệt vời! Video đã xuất bản 🎉'

💭 EMPTY STATES:
❌ 'No data' → ✅ 'Chưa có video nào. Tạo cái đầu tiên? 🎬'

⏳ LOADING:
❌ 'Loading...' → ✅ 'Đang phân tích... (15 giây)'
```

## SEO Content Strategy

```
1️⃣ KEYWORD MAPPING:
□ Primary keyword per page
□ Secondary keywords (3-5/page)
□ Long-tail cho blog

2️⃣ TOPIC CLUSTER:
   [Pillar Page]
       ├── [Cluster 1]
       ├── [Cluster 2]
       └── [Cluster 3]

3️⃣ META TAGS:
□ Title < 60 chars (keyword + benefit)
□ Meta description < 160 chars (CTA)
□ H1 unique per page
□ Alt text cho images

4️⃣ E-E-A-T SIGNALS:
□ Experience: Case studies
□ Expertise: Author bios
□ Authority: Backlinks, awards
□ Trust: Reviews, security badges

5️⃣ CONTENT CALENDAR:
│ Week │ Topic │ Keyword │ Type       │
│ 1    │ [...]│ [...]   │ How-to     │
│ 2    │ [...]│ [...]   │ List       │
│ 3    │ [...]│ [...]   │ Case Study │
│ 4    │ [...]│ [...]   │ Comparison │
```
